# ☕️ Morning LeetCode

[English](./README.md)

---
ソフトウェアエンジニアとしての基礎体力向上のため、毎朝30分（朝起きれたら）アルゴリズム学習を継続するプロジェクト。

### 📖 背景
実務での計算量への意識や、データ構造選定の「根拠」の強化が目的。単なる正解を目指すのではなく、「なぜその解法を選択したのか」の言語化を重視。

- **目標**: 3ヶ月以内に [Blind 75](https://neetcode.io/practice) 完走。

### ☕️ 毎朝のルーティン
1回30分のタイムボックス制

1. **自力で考察 (5分)**: 大体ここで静かに詰まる。
2. **AIからヒント (3分)**: 答えを薄目で見つつ、ヒントだけを摂取。
3. **実装 (10分)**: 脳内設計をコードに落とし込む。
4. **AIによる分析 (5分)**: 改善点や計算量をレビュー。
5. **(Option) 動画で別解 (5分)**: 解説動画を見て「なるほど」と呟く。
